Plataformas y herramientas

Diseñamos y desarrollamos plataformas de e-learning, sistemas de gestión (LMS) y experiencias inmersivas con AR/VR para aulas. Nuestra arquitectura prioriza escalabilidad, seguridad y facilidad de uso para estudiantes y profesorado.

¿Qué sistemas operativos se requieren?

  • ESO: Linux → porque es gratis y suficiente para lo que necesitan.
  • Bachillerato, SMX y Secretaría: Windows 10 → porque es el estándar en empresas y necesario para software más profesional.
  • Motivo general: Windows es el más usado, Linux es libre y sin coste.

Paquetes ofimáticos

ESO, Bachillerato y Secretaría

  • LibreOffice → gratuito, fácil de instalar y cubre lo básico: textos, diapositivas y hojas de cálculo simples.

SMX

  • Microsoft Office 365 (educativo) → total compatibilidad con documentos oficiales, Excel y PowerPoint más completos, incluye OneDrive y prepara mejor para el mundo laboral y universitario.

Sistemas de almacenamiento en la nube

  • ESO, Bachillerato y SMX: Google Drive → para guardar y compartir archivos fácilmente desde cualquier dispositivo con internet..

Software de edición gráfica, vídeo y música

  • ESO, Bachillerato, SMX y Secretaría:
    GIMP → edición de imagen
    OpenShot → edición de vídeo
    Audacity → edición musical

Software para virtualizar máquinas

  • SMX: Sí lo necesitan para practicar con distintos sistemas operativos sin riesgo.
  • ESO y Bachillerato: No es necesario, pero se instala por si en el futuro lo requieren.
  • Software elegido: VirtualBox → gratuito, fácil de usar y compatible con muchos sistemas.

Software para emular instalaciones y configuraciones de red

  • SMX: Necesario para practicar redes sin hardware físico.
  • ESO y Bachillerato: No es necesario, pero se instala por si lo necesitan más adelante.
  • Software elegido: Packet Tracer → gratuito, sencillo y perfecto para aprender redes.

Plataforma educativa del instituto

  • Todos los alumnos: Google Classroom → fácil de usar, organiza tareas y mejora la comunicación entre alumnos y profesores.
