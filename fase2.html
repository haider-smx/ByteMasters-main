Hardware

El mejor plan que hemos pensado para presupuesto bajo

Hemos decidido usar en secretaría, dirección y profesorado el ordenador de escritorio HP EliteDesk 800 G1, ya que no necesitan movilidad. Para ESO, SMX y Bachillerato hemos optado por portátiles, porque pueden necesitar moverse por el aula para realizar trabajos.

  • Para ESO y SMX: portátil PINSTONE N4000 ANL5.
  • Para Bachillerato y SMX: portátil Dell Latitude 5480.

El mejor plan que hemos pensado para presupuesto medio

Hemos optado por usar en secretaría, dirección y profesorado la PC estática HP EliteDesk 800 G1, ya que cumple con sus necesidades y no requieren movilidad. Para ESO, SMX y Bachillerato seguimos apostando por portátiles para facilitar el movimiento dentro del aula y el trabajo colaborativo.

  • Para ESO y SMX: portátil PINSTONE N4000 ANL5.
  • Para Bachillerato y SMX: portátil Dell Latitude 5480.

El mejor plan que hemos pensado para presupuesto alto

Hemos decidido mantener en secretaría, dirección y profesorado la PC estática HP EliteDesk 800 G1, ya que es suficiente para sus tareas diarias. Para ESO, SMX y Bachillerato se continúa utilizando portátiles para mejorar la movilidad dentro del aula y permitir trabajar en distintos espacios.

  • Para ESO y SMX: portátil PINSTONE N4000 ANL5.
  • Para Bachillerato y SMX: portátil Dell Latitude 5480.
